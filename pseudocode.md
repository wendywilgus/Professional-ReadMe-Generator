```
Create a README generator to quickly create a professional README for a new project
```

a command-line application that accepts user input

Prompt user for information about application repository

Use the following questions as a guide:

Make a list of questions:
What was your motivation?
Why did you build this project? (Note: the answer is not "Because it was a homework assignment.")
What problem does it solve?
What did you learn?
What makes your project stand out?

Set up inquirer with the questions

High-quality, professional README.md is generated with 
    - title of my project
    - sections: 
        - Description:  Prompt them for input for this paragraph/text
        - Table of Contents:  set list
        - Installation: what npm commands are needed to run application
        - Usage: What is the repo for?  How can it be used?
        - License: 
        - Contributing: How can others contribute to this repo?
        - Tests: what commands should be run to do tests?
        - Questions: How to ask questions about this repo

User enters project title --> displayed as the title of the README
User enters a description, installation instructions, usage information, contribution guidelines, and test instructions

This information is added to the sections of the README matching those sections

List of option for user to choose a license 
store selection
populate to generated READMe


Badge for that license is added near the top of the README 

Notice is added to the section of the README entitled License that explains which license the application is covered under

User enters GitHub username
Populate to the section of the README entitled Questions, with a link to GitHub profile

User enters email address
Populate to the section of the README entitled Questions, with instructions on how to reach me with additional questions

Links in the Table of Contents connect to the corresponding section of the README
